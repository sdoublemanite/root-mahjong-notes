# Root Mahjong Notes

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。页面内容主要围绕技术笔记、工具演示、学习记录等主题，以静态 HTML 文件形式存放，便于直接浏览或离线使用。

本仓库不针对任何特定域名或网站，仅为个人或协作过程中产生的 HTML 页面提供统一的存储和版本管理。

## 目录结构说明

```
root-mahjong-notes/
├── README.md          # 本说明文件
├── index.html         # 可选：仓库首页导航
└── pages/             # 存放各个独立 HTML 页面
    ├── page-001.html
    ├── page-002.html
    └── ...
```

- `pages/` 目录下为各独立页面文件，命名方式通常为 `page-xxx.html`，具体含义见文件内部注释或标题。
- 根目录下的 `index.html` 可根据需要提供页面列表或导航功能，非必需文件。

## 页面归档说明

- 每个 HTML 文件均为独立页面，不依赖本仓库其他文件（公共资源除外）。
- 页面可能包含内嵌 CSS 和 JavaScript，或引用外部 CDN 资源（如字体、图标等）。
- 页面内容以技术学习、笔记整理、小型工具演示为主，不包含商业、营销或诱导性内容。
- 部分页面可能处于草稿或实验状态，不代表最终版本。

## 使用方式

- 直接通过 GitHub 的页面预览功能查看任意 HTML 文件（需在仓库设置中启用 GitHub Pages）。
- 或将仓库克隆到本地，用浏览器打开 `pages/` 下的 HTML 文件即可。
- 如需自建导航，可参考 `index.html` 的格式编写列表。

## 维护说明

- 本仓库不定期更新，新增页面会放入 `pages/` 目录，并可能更新根目录的导航文件。
- 欢迎通过 Issue 或 Pull Request 提出修正、建议或新增页面。
- 提交的 HTML 页面应保持独立、自包含，避免引用本仓库外未公开的资源。
- 请勿提交包含敏感信息、违法内容或侵犯他人权益的文件。
